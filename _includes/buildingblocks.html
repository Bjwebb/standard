Activity

An activity is something that can be reported on through this standard. Activities can be grants, aid projects, investments or many other kinds of projects or collections of people, places and events which someone may wish to describe.

Types: Grant, Activity

Classification

A classification is used to describe an activity, document, transaction, organisation or other building block, usually using some pre-defined list of categories or tags. Classifications have code values and a vocabulary (which determines the 'code list' that code values should be found on). They may also include text descriptions and notes about the classification.

Types: Classification

Transaction

A transactions may be attached to an activity, and have a date, type, currency and value. Transactions may be commitments to fund an activity, disbursements of funding, or any other kind of financial transaction.

Types: Application, Commitment, Disbursement
